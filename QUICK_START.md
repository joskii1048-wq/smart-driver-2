# 🚀 Быстрый старт — Умный Водитель 2.0

## ✅ Что готово (MVP 100%)

### Полностью реализовано:

1. **GPS трекинг** — geolocator + permission_handler
2. **База камер** — 7 камер на маршруте Тутаев → Москва
3. **Физика торможения** — все формулы оптимальной скорости
4. **Умный собеседник** — 20 анекдотов + 6 историй + 10 советов
5. **TTS (Text-to-Speech)** — flutter_tts для голосовых фраз
6. **Настройки** — полная кастомизация (9 параметров)
7. **Демо-режим** — тестирование без GPS
8. **Режим "100 км/ч"** — жёсткий контроль скорости

## 📦 Что нужно для сборки APK

### Вариант 1: На вашем компьютере (Windows/Mac/Linux)

**Требования:**
- Flutter SDK ≥ 3.0.0
- Android Studio + SDK
- JDK 11+

**Шаги:**


### Вариант 2: На сервере (DigitalOcean VPS)

**Проблема:** На сервере нет Flutter SDK.

**Решение:** Установить Flutter на сервер.

```bash
# Установка Flutter на Ubuntu
cd /opt
wget https://storage.googleapis.com/flutter_infra_release/releases/stable/linux/flutter_linux_3.16.0-stable.tar.xz
tar xf flutter_linux_3.16.0-stable.tar.xz
export PATH="$PATH:/opt/flutter/bin"

# Проверка
flutter doctor

# Сборка APK
cd /home/openclaw/.openclaw/workspace/projects/smart-driver-2
flutter pub get
flutter build apk --release
```

**⚠️ Внимание:** Сервер имеет 2 vCPU + 4GB RAM. Сборка может занять 30-60 минут!

## 🎯 Что проверить после установки APK

### 1. Разрешения

При первом запуске приложение запросит:
- ✅ **Местоположение** (обязательно) — для GPS
- ✅ **Фоновое местоположение** (рекомендуется) — для работы в фоне
- ⚠️ **Поверх других окон** (опционально) — для overlay виджета

### 2. GPS

- Включите GPS на телефоне
- Нажмите кнопку "Старт" на главном экране
- Скорость должна обновляться в реальном времени

### 3. Умный собеседник

- Нажмите кнопку "🗣️ Собеседник"
- Должно прозвучать приветствие
- Каждые 3 минуты — новая фраза (анекдот/совет)

### 4. Демо-режим (если GPS не работает)

- Нажмите кнопку "Демо"
- Скорость зафиксируется на 120 км/ч
- Советник покажет рекомендации

## 🐛 Что делать, если что-то не работает

### GPS не работает

- Проверьте: включён ли GPS на телефоне
- Проверьте: разрешения в настройках Android
- Попробуйте: выйти на улицу (в помещении GPS слабый)
- Используйте: демо-режим для тестирования

### TTS не говорит

- Проверьте: громкость телефона
- Проверьте: настройки → "Голосовые подсказки" (должно быть ВКЛ)
- Попробуйте: перезапустить приложение
- Примечание: на эмуляторе TTS может не работать

### Собеседник молчит

- Проверьте: кнопка "🗣️ Собеседник" зелёная?
- Проверьте: настройки → "Умный собеседник" (должно быть ВКЛ)
- Подождите: первая фраза через 2-5 минут (зависит от частоты)
- Попробуйте: нажать кнопку "Собеседник" дважды (ВЫКЛ → ВКЛ)

### Сборка APK не запускается

- Проверьте: Flutter SDK установлен? (`flutter doctor`)
- Проверьте: Android SDK установлен?
- Попробуйте: `flutter clean && flutter pub get`
- Используйте: онлайн-сервис (Codemagic)
