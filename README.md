# Умный Водитель 2.0

**Кроссплатформенное приложение-навигатор с умным собеседником**

Android + iOS | Flutter 3.x | GPS + TTS + Физика торможения

---

## 🎯 Основные функции

### 1. ⚡ Интеллектуальный круиз-контроль
- Расчёт оптимальной скорости по физике торможения
- Учёт нештрафуемого порога (+20 км/ч, настраивается 0-30)
- Три типа панелей:
  - 🟢 **Зелёная** — можно разогнаться
  - 🟠 **Оранжевая** — рекомендуется снизить
  - 🔴 **Красная** — превышение!

### 2. 🗣️ Умный собеседник
- 20 анекдотов про водителей
- 6 историй из жизни
- 10 полезных советов
- 5 фраз подбадривания
- Система приоритетов (4 уровня)
- Мониторинг усталости (напоминание каждые 2 часа)

### 3. 📍 GPS мониторинг
- База камер на маршруте Тутаев → Москва
- Расстояние до ближайшей камеры
- Голосовые предупреждения (TTS)
- Фоновая работа

### 4. 🎛️ Режимы работы
- **Режим "100 км/ч"** — жёсткий контроль скорости
- **Демо-режим** — симуляция без GPS
- **Настройки** — полная кастомизация

## 📐 Формулы физики

### Оптимальная скорость

```
V_safe = V_camera + allowedExcess
```

Где:
- `V_camera` — ограничение скорости камеры (км/ч)
- `allowedExcess` — нештрафуемый порог (по умолчанию 20 км/ч)

### Расстояние торможения

```
S_need = (V_current² - V_safe²) / (2 × a)
```

Где:
- `V_current` — текущая скорость (м/с)
- `V_safe` — безопасная скорость (м/с)
- `a` — ускорение торможения (по умолчанию 2.0 м/с², настраивается 1.0-4.0)

### Рекомендуемая скорость

```
V_opt = √(V_safe² + 2 × a × S_actual)
```

Где:
- `S_actual` — фактическое расстояние до камеры (м)

## 🎯 Логика работы советника

### Алгоритм выбора панели

1. **Красная панель (CRITICAL)** — если `V_current > V_safe`
   - Прерывает любые фразы
   - Голосовое предупреждение: "ПРЕВЫШЕНИЕ! Тормозите до X км/ч!"

2. **Оранжевая панель (WARNING)** — если `V_current > V_opt + 5`
   - Прерывает разговорные фразы
   - Голосовое предупреждение: "Рекомендуется снизить до X км/ч"

3. **Зелёная панель (INFO)** — если `V_current < V_opt - 10`
   - Не прерывает другие фразы
   - Голосовое предупреждение: "Можете разогнаться до X км/ч"

### Система приоритетов фраз

| Приоритет | Уровень | Описание | Прерывает |
|-----------|---------|----------|-----------|
| 1 | CRITICAL | Превышение скорости | Всё |
| 2 | WARNING | Снижение скорости | Беседу |
| 3 | INFO | Разгон/проезд камеры | — |
| 4 | CONVERSATION | Анекдоты/советы | — |

## 🧪 Демо-режим

**Что можно тестировать без GPS:**

1. Симуляция скорости (фиксированная 120 км/ч)
2. Работа советника (все три панели)
3. Умный собеседник (анекдоты, советы)
4. TTS (голосовые подсказки)

**Как включить:**

Нажать кнопку "Демо" на главном экране.
