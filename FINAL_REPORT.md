## 🧮 Формулы физики (подробно)

### 1. Безопасная скорость

```
V_safe = V_camera + allowedExcess
```

**Параметры:**
- `V_camera` — ограничение камеры (км/ч)
- `allowedExcess` — нештрафуемый порог (0-30 км/ч, по умолчанию 20)

**Пример:**
```
V_camera = 90 км/ч
allowedExcess = 20 км/ч
V_safe = 90 + 20 = 110 км/ч
```

### 2. Расстояние торможения

```
S_need = (V_current² - V_safe²) / (2 × a)
```

**Параметры:**
- `V_current` — текущая скорость (м/с)
- `V_safe` — безопасная скорость (м/с)
- `a` — ускорение торможения (1.0-4.0 м/с², по умолчанию 2.0)

**Пример:**
```
V_current = 140 км/ч = 38.9 м/с
V_safe = 110 км/ч = 30.6 м/с
a = 2.0 м/с²

S_need = (38.9² - 30.6²) / (2 × 2)
       = (1513 - 936) / 4
       = 144 м
```

### 3. Оптимальная скорость

```
V_opt = √(V_safe² + 2 × a × S_actual)
```

**Параметры:**
- `V_safe` — безопасная скорость (м/с)
- `a` — ускорение торможения (м/с²)
- `S_actual` — фактическое расстояние до камеры (м)

**Пример:**
```
V_safe = 110 км/ч = 30.6 м/с
a = 2.0 м/с²
S_actual = 500 м

V_opt = √(30.6² + 2 × 2 × 500)
      = √(936 + 2000)
      = √2936
      = 54.2 м/с
      = 195 км/ч
```
