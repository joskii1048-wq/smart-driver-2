## 🔬 Формулы физики

### 1. Безопасная скорость

```
V_safe = V_camera + allowedExcess
```

**Пример:**
- Камера: 90 км/ч
- Порог: 20 км/ч
- V_safe = 110 км/ч

### 2. Расстояние торможения

```
S_need = (V_current² - V_safe²) / (2 × a)
```

**Пример:**
- Текущая скорость: 140 км/ч = 38.9 м/с
- Безопасная: 110 км/ч = 30.6 м/с
- Замедление: 2.0 м/с²
- S_need = (38.9² - 30.6²) / (2 × 2) = 289 м

### 3. Оптимальная скорость

```
V_opt = √(V_safe² + 2 × a × S_actual)
```

**Пример:**
- V_safe: 110 км/ч = 30.6 м/с
- Замедление: 2.0 м/с²
- Расстояние до камеры: 500 м
- V_opt = √(30.6² + 2 × 2 × 500) = √(936.36 + 2000) = 54.2 м/с = **195 км/ч**

## 🧪 Сценарии тестирования

### Тест 1: GPS трекинг

1. Разрешить доступ к местоположению
2. Выйти на улицу (GPS не работает в помещении)
3. Скорость должна обновляться каждую секунду
4. Проверить: точность координат

### Тест 2: Умный советник

1. Включить демо-режим (скорость 120 км/ч)
2. Приблизиться к камере (симуляция)
3. Должны появиться:
   - 🟢 Зелёная панель: "Можете разогнаться..."
   - 🟠 Оранжевая панель: "Рекомендуется снизить..."
   - 🔴 Красная панель: "ПРЕВЫШЕНИЕ!"

### Тест 3: Умный собеседник

1. Нажать кнопку "🗣️ Собеседник" (должна стать зелёной)
2. Должно прозвучать приветствие (TTS)
3. Через 2-5 минут — анекдот/совет
4. Проверить: прерывается ли беседа при превышении скорости

### Тест 4: Режим "100 км/ч"

1. Нажать кнопку "100 км/ч" (должна стать красной)
2. При скорости > 100 км/ч — красная панель
3. При скорости < 90 км/ч — зелёная панель
4. Проверить: игнорируются ли камеры

### Тест 5: Настройки

1. Изменить "Допустимое превышение" (0-30)
2. Изменить "Комфортное замедление" (1.0-4.0)
3. Включить/выключить "Голосовые подсказки"
4. Изменить "Частота подсказок" (2/3/5 мин)
5. Проверить: сохраняются ли настройки после перезапуска
